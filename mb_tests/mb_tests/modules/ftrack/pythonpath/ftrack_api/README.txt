Download the API package:
1. Go to yourstudioname.ftrackapp.com site.  
2. Go to your account icon in the upper right > System Settings
3. Download the API package
4. Extract the contents of the python-api.tar into this folder. It should look like this:
[root_dir]
	-modules
		-ftrack
			-pythonpath
				-frack_api
					-clique
					-requests
					ftrack.py
					FTrackCore.egg
					pyparsing.py
					six.py
					websocket.py
